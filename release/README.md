# Release Package

This directory contains the Step 3B submission bundle assembled from the frozen ranking tag plus the final packaging artifacts.

## Contents

- `RELEASE_MANIFEST.json` and `RELEASE_MANIFEST.md`: hashes, commands, runtimes, validators, and local versus Docker comparison
- `final_submission/`: canonical final CSV plus validation and invariance reports
- `hashes/`: SHA-256 sidecar files for key release artifacts
- `screenshots/`: real Streamlit sandbox screenshots used in the deck
- `deck/`: presentation copies and deck notes
- `sandbox/`: sandbox launch notes and sample-data references
- `docker/`: Docker reproduction notes and summary artifacts

The canonical submission file is `final_submission/Team_loading.csv`.
